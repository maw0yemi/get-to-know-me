    • CMP408-IoT and Cloud Secure Development

    Description

    This module delves into system programming, hardware, and operating systems, with a focus on security implications. Students will explore low-level programming, hardware interfaces, kernel functionality, debugging techniques, and cloud and web technologies to develop secure and efficient systems.

    Aims

    To equip students with the knowledge and skills needed to design, develop, and analyze system-level software and hardware interactions while addressing associated security challenges. This module aims to foster a deep understanding of the architecture, tools, and technologies that underpin modern computing systems.

    Learning Outcomes

    At the end of this module, I was able to:

    1. Demonstrate an understanding of hardware components, interfaces, and secure communication protocols.
    2. Analyse operating system structures and kernel operations, focusing on security principles.
    3. Employ cloud platforms for secure application development and deployment.
    4. Apply networking and web technology principles to build secure systems and address protocol-specific challenges.
