CMP320-Advanced Ethical Hacking

Description

This module is designed to develop understanding, knowledge and techniques associated with the various offensive attacks targeting computer systems and to provide the student with the opportunity to undertake an individual project

Aims

The aim of this module is to provide the student with an understanding of how attacks are developed, performed and analysed. The student will also gain an understanding of the countermeasures that a company can take to minimise the effect of attacks

Learning Outcomes

At the end of this module, I was able to:

  1. Demonstrate an understanding of the methods and techniques to examine a vulnerable system and identify malicious code using various malware analysis techniques
  2. Design, implement and test scripts to solve problems in Cybersecurity, Networking or Digital Forensics
  3. Demonstrate a critical evaluation of an advanced security topic with an independent project
