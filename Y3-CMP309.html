CMP309-Software Development for Mobile Devices

Description

This module develops a critical understanding of mobile software development practices and limitations of mobile devices. Students will implement and evaluate the techniques used to develop mobile applications.

Aims

The aim of this module is to provide the student with: the ability to design, develop and critically evaluate software for the mobile platform.

Learning Outcomes

By the end of this module I was able to:

  1. Design and implement mobile software incorporating appropriate UI design, services and communication functionality for multi-device applications using native frameworks and libraries;
  2. Discuss the devices, designs, services, languages and security issues relevant to mobile application development.
  3. Use appropriate programming constructs and communication protocols to enable device connectivity.
  4. Critically appraise usability, accessibility and security in mobile applications, i dentifying and exemplifying good practice.
