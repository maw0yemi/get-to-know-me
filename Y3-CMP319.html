CMP319-Web Application Penetration Testing

Description

This module develops a critical understanding of advanced techniques used by Ethical Hackers to examine the security of web applications.

Aims

This module's aim is to provide you with a solid understanding of web application penetration testing, from the techniques and tools used by penetration testers and bug bounty hackers, to the communication skills necessary for effective client response.

Learning Outcomes

At the end of this module, I was able to:

  1. Analyse and critically evaluate techniques used to assess web application security.
  2. Critically evaluate specific countermeasures to advanced hacking techniques
  3. Apply knowledge and understanding of a web application security assessment to communicate results in a professional manner
