    • CMP416-Advanced Digital Forensics

    Description

    This module will look at the forensic investigation of a network and the challenges analysts face when investigating network traffic. The module will provide students with an understanding of the underlying network principles important to comprehending network evidence. At the end of the module, students will be able to perform an investigation with a comprehensive, systematic approach to discover and examine evidence from networking infrastructure.

    Aims

    The module aims to provide the student with an understanding of the sources of network evidence associated with a networked environment. It will focus on how to recover, analyse and present that evidence in a forensically sound manner.

    Learning Outcomes

    At the end of this module, I was able to:

    1. Critically appraise a network forensic investigation involving evidence from network sources with respect to the legal definitions of computer misuse.
    2. Devise an appropriate professional level plan for such a network forensic investigation and carry out this plan within a context of a specific scenario using appropriate digital forensic tools.
    3. Critically analyse and evaluate the results of a host and network based digital forensic investigation.
    4. Critically analyse the challenges faced by investigators with respect to forensic investigations involving host and network devices, including computer, router. firewall and honeypots.
